## What Broke

Two proof bugs surfaced and were fixed:

- **Domain inference priority:** Lyhna source initially landed in `/homesteadai-io`
  because "The Keep" matched before "Lyhna." Fixed by prioritizing Lyhna terms first
  and moving the concept to `/lyhna`.
- **Stale source path:** generated concepts cited emptied `/inbox` paths. Fixed by
  treating the raw copy as the durable source reference and keeping the original
  inbox filename only as intake metadata.

## Remaining Risks

- Domain inference is keyword-based. It is useful enough for v1 proof, but not
  magic. Ambiguous source files may need explicit domain hints later.
- Source-deletion retirement is explicit-status-only because `/_raw` is intentionally
  local-only. Missing raw files in a clean clone must not auto-retire concepts.
